# DevFlow

Speak your thoughts. Structure your intent.

DevFlow transforms developer speech into structured, LLM-ready JSON prompts.  
It bridges raw thinking and precise AI interaction.

---

🚀 Live Demo

[https://devflow-seun.onrender.com](#)

---

## ✨ Features

- 🎙 Voice-to-text transcription (OpenAI Whisper)
- 🧠 Structured JSON generation
- 🎛 Mode selector (Debug / Feature / Refactor / Architecture)
- 📊 Clarity score indicator
- 💡 Smart suggestions panel
- 📤 Export structured JSON
- 🎨 Animated UI with subtle motion design

---

## 🏗 Tech Stack

- FastAPI
- OpenAI API (Whisper + GPT)
- Tailwind CSS
- Vanilla JavaScript
- Render (deployment)
